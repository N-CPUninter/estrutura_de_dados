## Como estudar
Clone este repositório em sua máquina e abra os arquivos conforme seus tipos.

## Executando os exemplos

Para executar os exemplos fornecidos, certifique-se de ter o Python instalado em sua máquina. Em seguida, siga estas etapas:

1. Clone este repositório em sua máquina local.
2. Navegue até o diretório clonado através do terminal ou prompt de comando.
3. Execute o comando `python main.py`.

Você verá a saída dos exemplos.
